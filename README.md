# viagens--landing-page

TripMe – Agência de Viagem (Landing Page)

Projeto desenvolvido como parte do Projeto de Certificação 1 – Trilha 2 (HTML, CSS e JavaScript) do programa DEVstart.

O objetivo foi criar uma landing page para uma agência de viagens, utilizando HTML semântico, CSS e JavaScript para melhorar a experiência do usuário.

---

Visão Geral

A TripMe é uma landing page focada em apresentar destinos, a proposta da agência e dicas de viagem, utilizando navegação interna e layout responsivo.

O projeto segue boas práticas de:
- Semântica HTML
- Organização de código
- Experiência do usuário
- Acessibilidade básica

---

Funcionalidades

- Banner com imagem de destino e texto em overlay
- Navegação fixa com links internos
- Scroll suave entre seções
- Destaque do menu conforme a seção ativa
- Botão flutuante para voltar ao topo
- Efeitos de hover em links e botões

---

Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

Estrutura do Projeto

---

Descrição dos Arquivos

index.html  
Responsável pela estrutura da página, utilizando tags semânticas como nav, main, section, article e footer.

style.css  
Define o layout visual do site, incluindo cores, tipografia, efeitos de hover e estilização do botão de voltar ao topo.

script.js  
Responsável pela interatividade da página, incluindo scroll suave, botão de voltar ao topo e destaque automático do menu conforme a rolagem.

---

Como Executar o Projeto

Opção 1 – Localmente  
1. Clone o repositório:
2. Abra o arquivo index.html no navegador.

Opção 2 – Live Server (VS Code)  
1. Instale a extensão Live Server  
2. Clique com o botão direito no index.html  
3. Selecione Open with Live Server
